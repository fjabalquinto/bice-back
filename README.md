# TEST BICELABS

## Intalacion

```bash
npm install
```

## Uso

```bash
npm start
```
El proyecto se levantara en localhost:9000/api.

Las URL son:

localhost:9000/api/last

localhost:9000/api/values/:valor

localhost:9000/api/date/:value/:date
